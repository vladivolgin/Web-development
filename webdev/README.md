# Hotel Booking Application

## Overview

This project is a hotel booking application built with **Flask** and **SQLAlchemy**. It allows users to view, create, update, and delete hotel bookings.

## Requirements

- **Python 3.x**
- **Flask**
- **Flask-SQLAlchemy**
- **Flask-WTF**
- **SQLite** 

## Running the app

- **python3 app.py**
